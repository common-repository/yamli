== Description ==
Implements Yamli service to text areas and post title text field for admin section.
Also implements Yamli service to text areas on theme level


== Installation ==

1. Upload `yamli.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new post and you should see the Yamli icon to the top most right of the fields.
4. Click the icon and select "Write in arabic".

== Frequently Asked Questions ==

= Its not working locally for me? =

Make sure you are connected to the internet, since this plug-in triggers an online service.

= Its working fine everywhere but not the post text area? =

It only works if the Visual Editor is disabled
